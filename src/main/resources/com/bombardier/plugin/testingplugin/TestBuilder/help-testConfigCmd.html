Here you should type the shell script that will be executed on the Slave machines. The script will be executed from the "home" directory of the user on the remote machine.
Available variables:
$TEST_ENV

- Specifies the location of the test environment directory on the Slave machines.
$TEST_WS

- Specifies the location of the project workspace directory inside the Jenkins root directory.
$TARGET_SCRIPT

- Specifies the location of the target script.
$TEST_LIST

- Specifies the location of the Test Case List (auto generated for each Slave).
