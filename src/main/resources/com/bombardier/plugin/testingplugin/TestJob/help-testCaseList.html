Specifies the path to the Test Case list on the Master machine. It can be absolute path, path or relative to the "userContent" directory located in Jenkins root.

The test cases path specified in this list shall be relative to the the list file. Such as:

List file location: "~/userContent/lists/list.lst"
Test file location: "~/userContent/tests/test.exp"

Test reference in the list file: "../tests/test.exp"
